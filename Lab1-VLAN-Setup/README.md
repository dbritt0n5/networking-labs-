# Lab 1: VLAN Setup

## Objective
Configure VLANs and enable communication between devices.

## What I Did
- Created VLAN 10 and VLAN 20
- Assigned ports to VLANs
- Configured trunk link

## Issues I Ran Into
- Devices couldn’t ping each other

## How I Fixed It
- Configured inter-VLAN routing on multilayer switch

## What I Learned
- VLANs separate broadcast domains
- Trunking allows multiple VLANs over one link
